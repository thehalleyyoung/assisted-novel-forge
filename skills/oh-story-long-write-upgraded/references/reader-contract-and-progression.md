# 读者契约与终局储备

本文件是 `story-long-write` 关于读者契约、主角代理权（叙事能动性/主导性）、利益安全与阶段推进的权威参考文件。相关洞察由授权社区样本提炼而来；样本多数是问题帖/避坑帖，属于非统计共识，也不能替代具体题材、平台和对标书判断。

**安全边界**：本文件不得包含来源站点隐私数据、访问凭证、cookies、tokens、请求签名、私有标识符，或任何能反推出私域账号/请求环境的信息。

## 1. 核心概念

### 读者契约
- 读者契约 = 开篇、题材、主角配置、叙事口吻共同许诺的阅读快感。
- 写作前先确认：读者是来看主角解决问题、见证团队群像、看导师托举后辈，还是看制度合作建设？不同契约下同一情节风险不同。
- 题材边界先于单条禁忌：政府、宗门总部、跨国组织、多国公司等高层级机构一出现，可能开启制度/势力范围情节线；先检查本书是否从开篇承诺了这种玩法。

### 主角代理权：因果权 + 结算权
- 主角代理权首先看**因果权 + 结算权**，不看出场时长，也不要求主角永远亲自动手或垄断每个镜头。
- 主角不可替代首先体现为因果权：主角通过决策、布局、授权、不可替代的信息/资源，或主动选择关键后果，决定事情为什么发生、如何进入与怎样转向。
- 结算权 = 核心收益、认可、权力或关键后果依照既有承诺回到应得者。可以借力、组队、被救、接受指导或让他人执行局部动作，只要因果与结算没有被无声夺走。
- 职业/权力幻想题材尤其敏感：读者通常期待主角掌握资源、做出决策、承担或选择后果并拿回可见收益。
- 题材例外（直给爽文）：番茄战神/赘婿/打脸/都市异能等，主角亲自出手、当面碾压本身就是契约兑现，代理权不能拿来给主角划水、让下属封神；只有布局/权谋/经营/群像/导师题材才默认可幕后运筹、隔人兑现。

### 期待所有权
- 承诺的兑现/高光属于本卷或本单元已经许诺的主角或其他承诺所有者；配角可以执行局部动作，但不能无声夺走期待所有权。
- 群像、导师或制度合作可把期待所有权明确给团队成员/学生/机构角色；关键是预先承诺、按约结算，而非事后解释抢戏合理。

### 关键节点四问
大纲、正文与审查遇到关键节点时统一问；答案可以指向主角，也可以指向本卷已承诺的其他所有者，不要求主角亲自完成每个动作：
1. 谁决定事情为什么发生/如何进入？
2. 谁作出不可替代的关键选择？
3. 谁承担或选择关键后果？
4. 核心收益/认可/权力最终结算给谁？

### 利益安全与兑现归属
- 利益安全 = 读者相信自己追的核心卖点不会被没收、稀释、转赠或交给无关角色消化。
- 兑现归属 = 谁铺垫、谁承压、谁选择，谁就应拿到主要情绪回报；旁观者/配角反应可以放大爽点，不能替代主角爽点。
- 核心资产（金手指、关键资源、身份底牌、独家技术、关键证据）不得无交换地交出、暴露给外部机构或被上级接管。若必须交出，需要可见交换：权力、保护、资源、渠道、名分、债权、信息、阶段胜利或更大局面的主动权。

### 期待债
- 每次埋承诺（主角会反击、资源会升级、身份会揭开、感情会推进）都产生期待债。
- 期待债可以延期，但要付利息：阶段性进展、新信息、局势变化、情绪释放或更清晰的下一目标。
- 用解释、设定或机构流程拖延兑现，会增加合理化债务。

### 终局储备与推进节奏

“写无可写”的病根是两类**宏观资源**被提前打光，跟单章涨多少关系不大。管住这两类、放开单章密度，才既不透支又不憋屈。

**两类会打光的宏观资源（开书时定，逐卷解锁）**
- **终局底牌**：头号宿敌、终极真相/身世、金手指上限、身份/地位终点、核心情感的最终确定。一次性资源，用了就没。开书时列进 `设定/题材定位.md` 的「终局底牌与升级台阶」小节，每项标最早可解锁的卷/阶段（从终局倒推成卷级里程碑，一卷只解锁一个大里程碑）。
- **升级台阶**：境界/等级/地图/势力层级的剩余档数。开书时做一次台阶检查——体系档数 × 每档停留篇幅 ≥ 全书目标字数；台阶不够是设计病，设计期补（拉长体系/加地图层），不靠每章憋。敌人/目标/地图成**梯队**，逐级解锁，禁止越级提前秒顶级 boss / 解决终极目标。

**单章 / 单个剧情单元放开密度（爽感来源，不要压）**
- 每个剧情单元设 1 条**主推线**（战力线/资源线/身份线/关系线/信息线/地图线/制度线/势力线/事业线/情感确定性等升级线其一，承担本轮最大高潮），其余线按剧情自然给**战果**（轻触到大涨皆可）。**一战多得——一场大胜同时破境、缴获、收势力、涨声望、抱得美人——是允许且常见的好设计，该给就给。**
- 章级推进按平台/题材分档：快节奏（番茄/爽文）保留“每章要有可见事件或爽点推进”的下限；慢热/种田/情感文才放宽到“改变目标 / 风险 / 信息 / 关系 / 资源 / 身份 / 情绪立场七类状态之一即可”。低压/过渡章用小而可见的变化承接，不强求每章高潮。
- 高潮/兑现后可安排一段短暂低压，让读者先拿小而可见的收益/奖励，再进下一轮压力；不强制兑现后立即加危机。
- 可选工具：行动成本/成功代价。重要成功可带时间、关系、暴露、资源消耗或新敌意等成本；低压/过渡/纯奖励章不强制附加代价。

**推进过快判定（透支两问）**
1. 是否动用了本阶段还不该解锁的终局底牌（宿敌了结、终极真相、身份终点、金手指触顶）？
2. 是否让某条升级线逼近天花板、后面没台阶接？

任一为“是” = 真正的透支预警 → 回退或改纲；两者皆否，多线齐涨放行。这套判定是经验法则，档位与里程碑按本书题材、平台与对标调整，不是硬门槛。

### 合理化债务
- 解释成本用于说明机构响应/接管、保密、资源调动、跨区执法、组织协作等为什么成立。
- 检查：这些解释是否服务核心情绪？若解释本身吞掉爽点、代入或主角高光，就先缩短解释、改为可见交换，或把机构反应降级为旁观/阻碍/资源接口。

## 2. 制度/机构与旁观者设计

- 提到政府、宗门总部、跨国组织、多国公司等不天然有毒；它们只是高层级叙事开关。
- 国家合作 / 政府合作类故事可以算契约安全，前提是开篇已承诺制度合作，主角保持不可替代，并获得可见权力、保护、资源、收益或更高行动权限。
- 若本书承诺的是个人逆袭、职业掌控、独享金手指，外部机构接管核心问题通常需要强交换与主角主动选择。
- 旁观者/反应层适合做：震惊、误判、舆论扩散、资源报价、敌我站队、见证主角选择。避免让旁观者替主角复仇、替主角做最终裁决、替主角拿核心收益。

## 3. 风险等级

| 等级 | 定义 | 处理 |
|---|---|---|
| 契约安全 | 与开篇题材契约一致；主角仍拥有关键选择/不可替代贡献；收益归属清楚 | 可写，注意把交换和收益可视化 |
| 需补强 | 可能稀释主角高光或推进过快，但能通过铺垫、交换、成本、题材声明修复 | 写前补理由、补代价或调整为阶段性而非终局兑现 |
| 契约破坏 | 没收核心卖点；自愿让渡同时造成**长期净收益为负 + 主角控制/因果权下降**（尤其没有可见战略交换）；无关角色夺走期待所有权；核心资产无交换上交或被接管 | 修纲；战略性让步可用局部聚光灯/资源换取更大控制、未来收益、关系/制度权力、信息优势，或兑现已承诺的群像/导师高光，不按让渡本身判红灯 |

## 4. 好/坏例与例外

### 职业/权力幻想主角自愿让出聚光灯
- 红灯组合：让渡同时导致长期净收益为负、主角控制/因果权下降，且看不到战略交换；不是任何自愿让渡都算 契约破坏。
- 好：主角授权下属执行，但方案、关键资源、最终谈判和风险承担由主角掌控；或用局部聚光灯/资源换取更大控制、未来收益、关系/制度权力、信息优势。下属表现反过来证明主角识人/组织能力。
- 例外：若本书从开篇承诺“导师/经营组织/群像成长”，主角的高光可以是培养体系、选人眼光和关键兜底。

### 给出资源或核心资产
- 坏：主角把独家技术、金手指产物、关键证据无条件交给机构，换来口头感谢，后续由机构推进主线。
- 好：主角交换到牌照、保护、渠道、话语权、独占收益或更大舞台，并保留不可替代的控制点。

### 无关角色替主角复仇
- 坏：压迫主角多章的敌人，被路过强者或随机事故解决，主角没选择、没出手、没获得认知/资源/地位变化。
- 好：配角可以助攻或见证，但关键证据、布局、决断、公开裁决或最后反击由主角完成。
- 例外：悲剧、悬疑、群像战争可让他人完成局部动作，但要让主角获得新的核心任务、代价或认知推进。

### 提前打光终局储备（多线齐涨本身没问题）
- 坏：一个剧情单元把全书级底牌一次性烧光——解决头号宿敌、揭终极身世、坐实终局身份、金手指直接触顶，后面没大目标可写。
- 好：本单元让主角一战多得（破境+缴获+声望齐涨），但都在体系台阶之内、不碰终局底牌；宿敌只退一个梯队、真相只揭一层，大底牌留到后续卷偿还期待债。

### 合法群像 / 导师故事例外
- 群像契约：每个 POV 有自己的目标和偿还，主角不是唯一爽点来源；卷契约应写清谁在本卷拥有高光。
- 导师契约：导师主角的高光在设计训练、识破困局、关键兜底、承担更高层代价，而不是每次亲手收割。
- 制度合作契约：开篇即承诺国家/机构协作，主角以专家、指挥者、关键资源拥有者或不可替代执行者身份获得明确权力和收益。

## 5. 续接与换书债

- **换书债**：引入新地图、新机构、新能力、新敌人或新谜团，既要支付必要的铺垫成本，也会新增期待债；不能靠新鲜元素逃避旧承诺。
- **题材特定高风险**：在履约爽文/能力幻想中，主角反复因愚蠢或可避免的无能制造灾难，再由别人收拾，会持续损害因果权、控制感和结算信任；这是依题材契约判断的高风险，不是所有故事的绝对禁令。
