# 题材风格包：双男主 / 宿命羁绊

> 知乎盐选特色大类。先读 `short-craft.md` 通用底座，本包在其上覆盖该题材专属调性。
> 一句话定位：**核心张力不是男女三角，而是两个男人之间的羁绊与撕扯，靠宿敌 / 知己 / 救赎 / 亏欠四条线放电；情感浓度拉满，但落在兄弟情、宿命感、占有欲的层面。** 读者要的是「一冷一热的温差对撞」+「嘴上势不两立、底下命都交出去」的错位快感。

---

## 叙述腔调

第一人称主导，但和追妻的单向受害者不同：双男主的「我」经常是**关系里主动、强势或占有的那一方**（把弟弟养在身边的哥、把对方当狗玩的重生者、冷淡履约的契约丈夫），另一方的滚烫与卑微靠对话和动作反衬出来。

腔调建在**口是心非**上：想的和说的相反，嘴上刻薄、心里在意。刻薄内心吐槽（「沈洲不用异能——可能是脑子有病」）和冷叙述（「对于他的死，我也没有什么感觉」）都高频。开口常带同性羁绊的称呼锚点：「我哥」「继弟」「死对头」「宿敌」「暗卫十七」。

允许**双视角 / 暗恋视差插叙**：在「我」的叙述里，穿一段另一方隐秘的付出（偷偷记的本子、透支换来的巧合、未送出的信），读者比「我」先知道对方爱得多深。这是本题材主引擎，不是上帝腔。

---

## 开篇范式

同样先放**结果倒序的黄金简介**（150-300 字），但剧透的不是背叛全貌，而是**两人错位的关系底牌**：谁在暗恋、谁欠了谁、谁会毁掉谁、火葬场追悔预告。正文第 1 章把这层错位慢放。

| 开篇式 | 真实范例 |
|--------|----------|
| 死讯倒叙 + 不解 | 《阴天快乐》谢疏玉死在我和他结婚后的第三年。自杀。 |
| 互殴定敌 + 埋暗流 | 《死对头暗恋我》我和沈洲是死对头，末世前针锋相对，纠缠不清。 |
| 占有宣言（哥 / 继弟） | 《我与我哥》我讨厌我哥。讨厌他用身子去换不干净的钱。 |
| 契约枷锁开场 | 《落网》沈则，三年后你想走，我放你走。可三年后我才知道，他从来没打算放我走。 |
| 重生 / 穿书预知死局 | 《双生劫》胎穿成将军府独子，我知道那个与我没有血缘关系的兄长，未来会因我一句话屠尽皇城。 |
| 沙雕反差钩 | 《疯子皇帝和老实暗卫》皇帝招暗卫，五十银两，包吃包住。可我领到的装备是透明薄衣和水晶胸链。 |
| 亏欠倒叙 + 确诊反击 | 《废弃玩物》谢寒舟是四九城里最不可攀的高岭之花。而我是他养在身边的继弟。可我却在确诊的那天，当着他的面，带了个野男人回家。 |

## 钩子母题与密度

密度高，但升级线不是「背叛升级」，而是**两人关系的锁扣一层层收紧**：从对峙到被迫共处，从共处到互相亏欠，从亏欠到谁也走不了。每节至少推进一次温差放电或一次视差揭穿，章尾留错位钩。

- **暗恋视差**（头号母题，共现「暗恋」576 次仅次于「甜宠」）：表面恨 / 斗 / 淡漠，底下一方偷偷爱着。视差揭穿的那一刻是最大爽点。
- **契约 / 欠债 / 联姻强绑**：助手协议、欠条、联姻、暗卫合约，把两个本该分开的人硬拴在一起，制造无法逃离的共处。
- **救赎与被救赎**：一方在深渊里（脏钱、自毁、疯魔、末世透支），另一方伸手拉 / 被拉。共现「救赎」326、「治愈」310。
- **亏欠反转（追夫火葬场变体）**：一方把另一方当工具 / 漠视 / 背叛，被弃者决绝离场（去父留子、带资产、嫁人），漠视者卑微追悔。
- **宿命预知**：重生 / 穿书知道对方未来会自杀 / 黑化 / 毁掉自己，主角逆天改命。共现「重生」68、「穿书」113。
- **身份反差硬碰**：高岭之花×舔狗继弟、皇帝×老实暗卫、将军×狼崽兄长、双 A / 双强互不相让。
- **信物 / 记录承重**：偷记的本子、未送出的信、唯一的合照、淡去的标记，把没说出口的感情具象成物件。

## 情绪烈度与模式（烈度 8）

不是追妻那种一味直给宣泄，而是**双轨温差**：一条线冷、嘴硬、克制、口是心非；另一条线滚烫、卑微、偏执、豁出去。两条温差对撞放电，是本题材的情绪引擎。BE 罕见（仅 56），主流走 HE / 救赎（HE 共现 1054），别写死写绝。

- **冷轨**：直写「没有感觉 / 不在意 / 只觉得烦」，越平越反衬后面的痛。《阴天快乐》对于他的死，我也没有什么感觉。
- **热轨**：卑微、偏执、追悔的滚烫台词，独立成段砸。《喵的，栽你手上了》哥，我错了，你别不要我。
- **别扭轨**：傲娇的口是心非，想的和说的相反，嘴上越刻薄越藏着在意（如《死对头暗恋我》里对沈洲每次的挑衅）。
- **反差痛**：把冷叙述和一个滚烫细节并置，越平的口吻托越重的内容。《阴天快乐》我像对待一件再正常不过的工作一样去处理这件事情。冷静地给他挑遗照，选墓地。

真实范例：

- 《死对头暗恋我》也不知道，沈洲每一次嘴里的巧合，都是他异能透支换来的结果。
- 《双生劫》他昏迷中死死攥着我手腕呢喃：再…再骂响些…
- 《与君朝》他留给我的，只有无数张未送出的信。每一封信，都写满了他从未对我说出口的感情。

## 对话风格与真实金句

占比高，独立成段。三类台词撑全篇：

**① 嘴硬 / 攻击性对手台词（表面势不两立）**
- 好看吗？死变态！
- 他说有你就信啊。我还不知道他，一天天娘们唧唧没个男人样，被人碰一下就哭。
- 蠢货！练枪要扎马步！

**② 卑微 / 偏执的滚烫（舔狗 / 病娇 / 追悔）**
- 哥，我错了，你别不要我。孩子是谁的？我给孩子当爹好不好？
- 我爱你，离不开你。
- 宝宝，留在我身边，你要多少钱都可以，你想让我穿什么都可以，你想踩哪儿都行。
- 再…再骂响些…

**③ 契约 / 立场台词（宿命枷锁）**
- 沈则，三年后你想走，我放你走。
- 可以，但是你要写欠条。
- 你要多少钱都可以，随你开价，我都给你。

冲突不靠谩骂，靠**温差落差**标记权力与心动的易主：前期一方长篇挑衅、另一方冷淡两不搭理；后期冷的那个破防、硬的那个卑微。谁先服软，谁就露了心。

## 节奏骨架

```
黄金简介（结果倒序 + 暗恋 / 亏欠 / 宿命的关系底牌剧透）
→ 第1章互殴 / 契约 / 占有场景慢放，立敌意 + 埋暗流
→ 关系锁扣收紧（被迫共处 / 契约 / 宿命枷锁）
→ 温差一次次放电（嘴硬 vs 滚烫，错位加压）
→ 视差 / 隐情揭穿（本子 / 透支 / 未送出的信曝光）
→ 危机或亏欠爆点（送死 / 背叛 / 漠视 / 黑化）
→ 双向奔赴 或 追悔火葬场
→ HE 收束（宿命和解 / 救赎闭环）
```

每章一个新错位或新揭穿，章尾留钩不收束。疏密：温差放电、视差揭穿、危机爆点处详写慢放，共处日常一句带过。

## 收尾范式

- **双向奔赴**：嘴硬的一方终于承认，冷轨破防，温差归零。
- **救赎闭环**：把对方（或被对方）从深渊拉出，从「两不相欠」走到「两相依」。
- **追悔火葬场（亏欠子类）**：漠视者卑微追回，被弃者冷拒或有条件回头。
- **宿命反写（重生 / 穿书）**：改掉前世的死局，这次不走了 / 你不许死。
- 主流是 HE，别学追妻那样纯虐离场；就算前面虐到底，也要给和解或圆回。

情节概述：《所幸我们总会相逢》季寻不告而别的第三年，一张唯一的合照牵出重逢。／ 《与君朝》重生逆写萧翊熬死的结局，把人囚在身边补回没说出口的感情。

## 该题材专属调校（与通用去 AI 规则的偏差）

这些在别的语境像 AI 味或用力过猛，在本题材是卖点，**别误删**：

- **双视角 / 暗恋视差插叙**：在「我」的叙述里剧透另一方的隐秘付出，保留。这是共鸣主引擎，不是上帝腔。只删「泄了底又不吊读」的随手全知。
- **口是心非的傲娇内心独白**：想的和说的相反，保留，别抹平成前后一致。
- **卑微 / 偏执 / 追悔的滚烫台词**：舔狗、病娇、火葬场的告白，保留，别当「用力过猛」删。
- **契约 / 宿命的前瞻剧透**：有意识地向前吊「三年后 / 后来他后悔了 / 未来他会毁掉我」，保留。

仍要避免的真 AI 味：映入眼帘、心中暗道、沉声道、嘴角微扬、目光如炬、论文体连词（不难看出 / 由此可见）、空泛无体感的情绪总结句（一丝悲伤涌上心头）。语料里也偶有「映入眼帘」这类混入，别照抄。

## 配套结构公式（按篇幅选）

- **宿敌暗恋 / 双向奔赴型（8-11 章）**：1-2 章互殴立敌 + 埋暗流 → 3-4 章被迫共处 / 契约绑定 → 5-6 章视差揭穿（一方隐秘曝光） → 7-8 章危机送命 / 破防表白 → 双向奔赴 HE。视角可单可双；温差是引擎，冷的那个越晚破防越好。
- **救赎 / 现代情感型**：一方在深渊（脏钱 / 契约 / 自毁 / 末世透支） → 另一方拉拽 → 前世 / 契约枷锁把人锁死 → 打破枷锁走到两相依。冷叙述打底，反差细节扎心。
- **亏欠追夫火葬场型**：一方漠视 / 背叛（继弟 / 伴侣被当工具） → 确诊 / 怀孕 / 心死 → 决绝离场（去父留子 / 带资产 / 嫁人） → 漠视者卑微追回 → 冷拒或有条件回头。刀口精准，别谩骂。
- **古代权谋双强型**：帝王将相 / 兄弟养成，重生或穿书预知死局 → 逆写宿命 → 囚 / 救 / 亲 → 江山与人都要。权谋线和感情线互为筹码。
