# More than the sum of its parts

## Failure mode of remixes
Installing N skills and listing them in SKILL.md produces **subadditive** behavior: context thrash, contradictory advice, humanizers that flatten voice, and false-done from the friendliest member.

## Superadditive mechanisms in this package
1. **forge_state.json** — single pressure vector
2. **compose.py** — budgeted prescription from pressures
3. **adversarial pairs** — detect family ∩ scrub family = ∅
4. **voice_contract** — hard constraint on scrubbers
5. **forge_brief.md** — agent-facing minimal work order
6. **compound_pressure in verify_gate** — emergence metric as done criterion

## Metric
`compound_pressure` falls only when *coupled* problems clear — scrubbing tells without fixing continuity still leaves pressure high.
